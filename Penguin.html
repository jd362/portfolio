Penguin VR Project

Penguin VR project is an educational game project developed in Unity and deployed on Oculus Quest2 VR platform.

This game designs an immersive role-play experience for the player. The player will act as a penguin and perform mini games that simulate the daily activities of a penguin: laying eggs, defending the nest, attracting mates, etc.

This is the first project I participated in the WID Virtual Environment Group. In this project, I worked on the programming part of the game, including:

Building the mini-games in Unity, and scripting using C#.

Implementing a gesture-based control system. Utilizing the hand detection system of Quest2 VR headset, the game allows player to interact with the game using hand gesture and body movement, without relying on controller.

Polishing the game.
