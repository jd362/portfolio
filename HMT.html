Dice Adventure

Dice Adventure is the first project I was involved in CMU TACT team. It is a multi-player turn-based RPG game. 3 players will cooperate with each other, defeat monsters, and adventure trough the maps.

In this project, I worked to create a local mode for the game, which allows an AI-friendly game mechanics. We will use this game to explore methods that facilitate AI to cooperate with human players.
